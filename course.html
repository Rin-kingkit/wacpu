專案學生 / 教師認證課程

LEVEL 1

本課程將實際搭配多種生活範例，引領初學者入門，無論是跨裝置互動控制、環境數據集、儲存至資料庫皆不成問題，通過認證考試後，將具備基礎的運算思維的概念，並有資格成為Webduino 助理講師的一員。

課程特色
  • 原廠授權之課程、教材
  • 12 小時精實的物聯網課程培訓
  • 帶走物聯網基礎開發套件組
  • 了解運算思維概念與應用 ( 變數、邏輯、迴圈...等 )
  • 多元控制的豐富範例 ( 多台連動、結合網路與行動裝置應用 )
  • 課後考試認證通過頒發證書，並具備「官方助理講師資格
  • 包含午餐
課程材料包

Dr. Smart 開發者套件包

考試方式
  • 通過學科考試
  • 完成術科考試
  • 評鑑上台試教狀況

認證考試

官方學習歷程認證
( Level 1 / Level 2 )

認證對象：
  • 在學學生、自學團體、有興趣之教學工作者
考試內容：
  • 如上相關課程之內容，包含筆試、術科、現場實地操作。
考試時間：於官網公布
考試地點：Webduino 官方原廠 / 慶奇科技 ( 高雄 )

官方講師認證

認證對象：
  • 已經通過 Level 2 之有興趣從事長期教學之工作者
  • 願意推廣 Webduino 相關領域之研究、教學、課程教案顧問者
考試內容：
  • 如上相關課程之內容
  • 實際跟課助教 20 小時，現場實際教學 40 小時，並採取樣錄影。
考試時間：於官網公布
考試地點：Webduino 官方原廠 / 慶奇科技 ( 高雄 )
