專案學生 / 教師認證課程

LEVEL 1

本課程將實際搭配多種生活範例，引領初學者入門，無論是跨裝置互動控制、環境數據集、儲存至資料庫皆不成問題，通過認證考試後，將具備基礎的運算思維的概念，並有資格成為Webduino 助理講師的一員。

課程特色
  • 原廠授權之課程、教材
  • 12 小時精實的物聯網課程培訓
  • 帶走物聯網基礎開發套件組
  • 了解運算思維概念與應用 ( 變數、邏輯、迴圈...等 )
  • 多元控制的豐富範例 ( 多台連動、結合網路與行動裝置應用 )
  • 課後考試認證通過頒發證書，並具備「官方助理講師資格
  • 包含午餐
課程材料包
  • Dr. Smart 開發者套件包
考試方式
  • 通過學科考試
  • 完成術科考試
  • 評鑑上台試教狀況

LEVEL 2

本課程延續 Level 1 運算思維概念，同步加強程式邏輯訓練，並加入更多樣化的傳感器與程式邏輯判斷。透過多種生活範例的實作，能更精的實加強程式訓練，強化立刻動手解決生活中的問題的行動力，通過認證考試將具備一定程式能力，同時擁有報考官方講師之資格。

課程特色
  • 原廠授權之課程、教材、考試
  • 24 小時（12 小時課程，12 小時實際教學 ) 精實的物聯網課程培訓
  • 帶走物聯網進階開發套件組
  • 活用程式邏輯概念與深入應用 ( 陣列、函式 )
  • 深入學習綜合型範例 ( 雲端資料庫、語音聲控及朗讀 )
  • 完整程式教學快速指南
  • 課後考試認證通過頒發證書，並具備「報考」官方講師資格
課程材料包
  • 智慧植栽材料包
  • 智慧插座擴充套件
考試方式
  • 通過學科考試
  • 完成術科考試
  • 評鑑上台試教狀況

專案學生 / 教師認證課程

本課成為各級教育單位客製化合作之認證課程，有專屬之證書、內容、效期、學習指標等。例如：學校社團、冬夏令營隊、短中長期之培訓計畫、教師公民營等。需另案與認證中心洽詢。

課程特色
  • 客製化課程，方便配合作單位設計與其相關之特色教學規劃
  • 時間較短，適合入門初學及跨領域設計
